How to Uninstall MotionSaver üì¥

  1. Open Task Scheduler:
    • Press Win + R to open the Run dialog.
    • Type taskschd.msc and press Enter.
    • Alternatively, search for Task Scheduler in the Start menu and open it.
  2. Find the MotionSaver Task:
    • In the left pane, click on Task Scheduler Library.
    • Look for a task named PhotoEngineAdminTask in the list.
  3. Disable the Task:
    • Right-click on PhotoEngineAdminTask and select Disable.
    • This will prevent MotionSaver from starting automatically.
  4. Stop MotionSaver:
    • Open StopMotionSaver.bat from your C:/MotionSaver or D:/MotionSaver folder to immediately exit the screensaver and stop all related processes.
  5. Run the Uninstaller:
    • Locate and run Uninstall.exe as Administrator from the same folder (C:/MotionSaver or D:/MotionSaver) to completely remove MotionSaver from your system.
  6. Remove Remaining Files (Optional):
    • If any files or folders remain after uninstalling, you can permanently delete them by selecting the folder (C:/MotionSaver or D:/MotionSaver), then pressing Shift + Delete to remove it completely.

You can re-enable the task at any time by right-clicking it and selecting Enable.

Frequently Asked Questions (FAQ) ‚ùì

The screensaver doesn't start with Win + S. What do I do?

Ensure the background service is running. You should see the MotionSaver icon in your system tray. If not, please re-run MotionSaver.bat as an administrator.

How do I change the default password?

Run Gui.bat as an administrator to open the settings GUI. You can update the password there. The default password is test. You should change it to something more secure.

What should I do if I get a black screen?

1. Ensure that the extracted folder structure is correct and all files are present. You should launch MotionSaver.bat from the D:/MotionSaver folder.
2. Press Esc to exit the screensaver.
3. Type test (the default password) to unlock.
4. If test does not work, press Ctrl + Alt + Del to bring up the Windows security screen and unlock your PC.

How do I change the video?

You can download a 1080p 30fps/60fps video and save it anywhere. Then, run Gui.bat as an administrator, use the option to change the video file, select your new video, and click Save. Make sure to restart the MotionSaver application for the changes to take effect.

My antivirus flagged MotionSaver. Is it safe? üõ°Ô∏è

Yes, MotionSaver is safe. Some antivirus programs may flag it due to the use of batch files and system-level hooks. You can add the D:/MotionSaver folder to your antivirus exclusion list to resolve this.

Some Windows shortcuts (like Alt+Tab) don't work. Why? ‚ö†Ô∏è

MotionSaver uses low-level keyboard hooks to secure the screen, which can block some system shortcuts. If you need to restore them, you can run StopMotionSaver.bat as an administrator. This is a known limitation that we are working to improve.

How do I send logs to the developers? üêõ

You can find the logs folder inside D:/MotionSaver. Please open an issue on the GitHub Issues page, describe the problem, and attach the relevant log files.

My FPS feels glitchy, like it's running below 30 FPS. What should I do?
  1. Try changing the video using the GUI as shown above in the "I want to change the video" section.
  2. Download another short 1080p video (about 10 seconds) from YouTube or any online source and test if the issue persists.
  3. If you still experience problems, please upload your log files under a new issue on GitHub Issues, including your Instagram handle (if applicable) and your Windows version. This will help the developers diagnose the problem more effectively.
I only have a C drive (no D drive). What should I do?

If you only have a C drive, you can still use MotionSaver by extracting the MotionSaver folder to C:/MotionSaver instead of D:/MotionSaver. Make sure the folder structure remains the same.

  • Move the extracted MotionSaver folder to your C:/ drive so the path is C:/MotionSaver.
  • Update any references to D:/MotionSaver in your settings or configuration files to C:/MotionSaver.
  • Proceed with the rest of the installation steps as described above.
  1. Try opening Gui.bat and update all folder paths from D:/ to C:/.
  2. If Gui.bat does not open, manually edit the config.json file in the config folder and change all paths to use the C:/ drive.
  3. After updating the paths, launch MotionSaver.bat from the new location on your C drive.
I am not able to close the lockscreen or the password prompt is not working. What should I do?
  1. Press Esc or try clicking anywhere on the lockscreen. This should display the password field. The default password is test.
  2. If the password prompt does not appear or is unresponsive, press Ctrl + Alt + Del. This will bring up the original Windows lockscreen. Enter your Windows password to unlock.
    • After unlocking, try entering the password (test) again in the MotionSaver prompt if it reappears.
    • Note: This issue can occur if the keyboard hooks are released unexpectedly.
  3. If neither of the above works, press Ctrl + Alt + Del and select the Sign out option. Then log in again to your Windows account.
Sometimes my Win + <key> shortcuts don't work. What should I do?
  • MotionSaver uses low-level keyboard hooks, which may affect shortcut responsiveness.
  • To use Win + <key> shortcuts (like Win + X for Quick Access or Win + R for Run), press and hold the Win key for at least 1 second before pressing the other key.
  • Example: Hold Win for 1 second, then press X or R to trigger the shortcut.
The clock is not visible, or I want to change the clock font. How can I do this?
  • If the clock is not visible, try downloading and installing the Minecraft PE font or any other preferred font from the web.
  • You can customize the clock, weather, and music fonts:
    1. Open Gui.bat as administrator.
    2. In the GUI, navigate to the font settings for clock, weather, or music.
    3. Download and install any external fonts you like; they will appear in the dropdown selection.
    4. Select your desired font and save the settings.
  • This allows you to personalize the appearance of the screensaver's text elements.
Where can I download 60fps 1080p videos for MotionSaver?
  1. Download yt-dlp:
    Visit the yt-dlp GitHub page and download the latest version of yt-dlp.
  2. Organize your videos:
    Place yt-dlp.exe in a folder such as D:/Live_wallpapers for easy access.
  3. Download 1080p 60fps videos:
    Open a command prompt in the D:/Live_wallpapers folder and run the following command (replace the YouTube link with your preferred video): 
    yt-dlp -f "bestvideo[height<=1080][fps=60][ext=mp4]+bestaudio[ext=m4a]/bestvideo[height<=1080][fps=60]+bestaudio" --merge-output-format mp4 https://www.youtube.com/watch?v=21JAe9vqb-I
  4. Download 1080p 30fps videos:
    Use this command for 30fps videos: 
    yt-dlp -f "bestvideo[height<=1080][fps=30][ext=mp4]+bestaudio[ext=m4a]/bestvideo[height<=1080][fps=30]+bestaudio" --merge-output-format mp4 https://www.youtube.com/watch?v=21JAe9vqb-I

    5. Tip: You can also download Instagram Reels or videos using yt-dlp! For example, to download a 1080p 60fps Instagram Reel, use: 

    yt-dlp -f "bestvideo[height<=1080][fps=60][ext=mp4]+bestaudio[ext=m4a]/bestvideo[height<=1080][fps=60]+bestaudio" --merge-output-format mp4 https://www.instagram.com/reel/DKRKBEnJVrS/?utm_source=ig_web_copy_link
    Replace the link with your desired Instagram video or reel URL.

After downloading, follow these steps to set the new video as your MotionSaver lockscreen background:

  1. Exit MotionSaver if it is running.
  2. Open Gui.bat.
  3. Select the downloaded video file.
  4. Click on Save.
  5. Start MotionSaver again. It will now use the new video as your wallpaper.

How do I change the wallpaper?
  1. Run GUI.bat as administrator.
  2. Select your desired screensaver video in the GUI.
  3. Make sure to exit the running MotionSaver application from the system tray before applying changes.
  4. Click on Save in the GUI.
  5. Run MotionSaver.bat as administrator to start with the new wallpaper.
How do I change the profile GIF?
  1. Prepare your GIF:
    Make sure your GIF is 400x400 pixels for best results.
  2. Open the GUI:
    Run gui.bat as administrator.
  3. Select your GIF:
    In the profile picture/GIF menu, click Browse and choose your GIF file.
  4. Crop the image:
    Click Crop Image. You can zoom in/out or drag the image to adjust the crop.
    • If cropping is buggy, simply click Circle Crop and proceed.
  5. Stop MotionSaver:
    Exit the running MotionSaver application from the system tray before applying changes.
  6. Save settings:
    Click Save Setting in the GUI.
  7. Restart MotionSaver:
    Run MotionSaver.bat as administrator to apply your new profile GIF.
